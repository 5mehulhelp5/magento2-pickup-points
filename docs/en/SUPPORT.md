# Innosend Pickup Points Module - Support

## Support Options

See Base module support documentation for support options and contact information.

## Common Issues

### Pickup Points Not Appearing

1. Verify Base module is configured correctly
2. Check API connectivity
3. Verify shipping address format
4. Check browser console for errors

### Map Display Issues

1. Verify internet connection
2. Check Leaflet library loading
3. Verify map container element exists
4. Check browser console for errors

## Reporting Issues

Include:
- Magento version
- Module version
- Browser and version
- Steps to reproduce
- Console errors
- Network request details
